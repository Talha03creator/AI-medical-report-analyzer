API Documentation

Full interactive documentation available at /docs (Swagger UI) and /redoc (ReDoc).

POST /api/v1/reports/upload

Upload and analyze a medical transcription (TXT/PDF)

curl -X POST http://localhost:8000/api/v1/reports/upload \
  -F "file=@report.txt"
GET /api/v1/reports

List all analysis history (paginated)

curl http://localhost:8000/api/v1/reports?page=1&per_page=20
GET /api/v1/reports/{id}

Get full analysis for a specific report

curl http://localhost:8000/api/v1/reports/{report-uuid}
GET /api/v1/reports/{id}/export

Download analysis as JSON file

curl -O http://localhost:8000/api/v1/reports/{id}/export
GET /api/v1/health

System health check (database, cache status)

curl http://localhost:8000/api/v1/health
